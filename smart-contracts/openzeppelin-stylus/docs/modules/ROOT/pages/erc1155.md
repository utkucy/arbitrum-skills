# ERC-1155

ERC1155 is a novel token standard that aims to take the best from previous standards to create a xref:tokens.adoc#different-kinds-of-tokens[**fungibility-agnostic**] and **gas-efficient** xref:tokens.adoc#but_first_coffee_a_primer_on_token_contracts[token contract].

[[erc1155-token-extensions]]
## Extensions

Additionally, there are multiple custom extensions, including:

* [ERC-1155 Burnable](erc1155-burnable.md): Optional Burnable extension of the ERC-1155 standard.

* [ERC-1155 Metadata URI](erc1155-metadata-uri.md): Optional extension that adds a token metadata URI.

* [ERC-1155 Pausable](erc1155-pausable.md): A primitive to pause contract operation like token transfers, minting and burning.

* [ERC-1155 URI Storage](erc1155-uri-storage.md): A more flexible but more expensive way of storing URI metadata.

* [ERC-1155 Supply](erc1155-supply.md): Extension of the ERC-1155 standard that adds tracking of total supply per token id.
